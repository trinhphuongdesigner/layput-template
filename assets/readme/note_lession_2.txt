[NỘI DUNG ĐÃ HỌC]

1. Bố cục website

2. Các thẻ HTML thông dụng: 
  - Thẻ định dạng văn bản: h1-h6, p, ul/ol - li, strong, u, em, small, sub, u, sup, br, textarea,...
  - Thẻ tiện ích: div, a, img, input, button, hr, br, label, select, option.
  - Thẻ ít dùng: iframe, audio.

3. Các phím tắt:
  - Sao chép đoạn code (ctrl + c).
  - Sao chép dòng code (shift + alt + mũi tên lên xuống).
  - Đảo vị trí dòng - đoạn code (alt + mũi tên lên xuống).
  - Tạo nhiều con trỏ: giữ alt + click hoặc giữ ctrl + alt + mũi lên lên xuống.
  - Trở về cuối dòng: end hoặc fn + end.
